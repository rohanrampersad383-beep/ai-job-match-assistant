# MatchIQ Branding Assets

This folder is the source-controlled brand asset home for MatchIQ.

- `logo-mark.svg`: compact product mark for navigation and app chrome.
- `logo-full.svg`: horizontal brand lockup.
- `favicon.svg`: small-format browser icon source.
- `app-icon.svg`: square app icon source for installable surfaces.

Public copies live in `/public` and `/public/branding` so Next metadata, favicons, and future social assets can reference stable URLs.
