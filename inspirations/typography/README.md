# Typography

## Inspiration Links

- Vercel Geist: https://vercel.com/geist/introduction

## Notes

- Typography should be sharp, minimal, modern, and premium.
- Preserve readable body text in dense product surfaces.

## Implementation Ideas

- Use shared typography components/classes for display, heading, body, label, caption, and mono text.
- Keep letter spacing at zero by default except intentional uppercase labels.

## Performance Considerations

- Use Next font loading and avoid excessive font families.

## Adaptation Ideas For MatchIQ

- Use display type sparingly for brand moments and tighter product typography inside panels.
