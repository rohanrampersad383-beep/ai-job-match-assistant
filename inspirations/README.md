# MatchIQ Inspiration System

This directory is the long-term visual research system for MatchIQ, a premium AI-powered career intelligence platform.

Use each subfolder to collect inspiration links, observations, implementation ideas, performance notes, and adaptation rules. Inspiration should guide the design language without copying another product directly.

Current foundation references:

- Motion.dev React transitions: https://motion.dev/motion/transition
- Motion.dev React animation guide: https://motion.dev/docs/react-animation
- Vercel Geist design system: https://vercel.com/geist/introduction
- Framer reduced-motion accessibility notes: https://www.framer.com/help/articles/reduced-motion-settings/

Adaptation principle: keep MatchIQ dark, sharp, graphitic, AI-native, and career-intelligence focused. Prefer reusable tokens and composable primitives over one-off visual treatments.
